# CV_StudyCode

一个专注于计算机视觉（Computer Vision）领域的学习与代码实践仓库。

## 仓库简介

本仓库旨在系统性地整理和记录计算机视觉相关算法的学习过程、代码实现和实验数据。内容涵盖了计算机视觉的多个核心任务，包括图像分类、目标检测、语义分割、实例分割、生成模型、多模态大模型等，适合初学者学习和参考。

## 主要内容

- **Image_Classification**: 包含经典的图像分类模型（如ResNet, VGG等）的实现、训练和评估脚本。
- **Image_ObjectDetection**: 包含目标检测模型（如YOLO, Faster R-CNN, SSD等）的代码实践。
- **Image_Segmentation**: 包含图像分割模型（如U-Net, FCN, Mask R-CNN等）的代码实践。
- **Dataset**: 用于存放项目所需的数据集。建议将大型数据集通过符号链接或引用方式放在此目录下。
- **DataSet_Links.md**: 文档中记录了各项目所用数据集的获取方式与官方链接，方便复现实验。
